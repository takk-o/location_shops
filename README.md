# location_shops

## Overview
指定した場所から半径1Kmの、指定キーワード(Ex.:カフェ、居酒屋、コンビニ、等)に該当する'店舗'情報を収集し、Excelに出力する。

## Requirements
- Python 3.11.7
- python-dotenv 1.0.1
- googlemaps 4.10.0
- openpyxl 3.1.2
- pyarrow 15.0.0
- pandas 2.2.0

## Usage
1. location_shops.pyを起動
1. 'location: 'に、場所を指定
1. 'keyword: 'に、キーワード(Ex.:カフェ、居酒屋、コンビニ、等)を指定
1. 'output/shopdata.xlsx'に店舗情報が出力される
